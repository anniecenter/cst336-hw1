C#

C# is a modern, object-oriented programming language developed in 2000 by Anders Hejlsberg and his team at Microsoft.

It is great for developing both web and desktop applications, and it is widely used for game development as well. One reason it is popular with game developers is that it works well with the Unity game development platform.

C#, Java, and C++ all have similar syntax, so if you know one, it is easy to learn the others.
