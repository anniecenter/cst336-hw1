Java

Java is an object-oriented programming language that was developed in 1995 by Sun Microsystems, but it has since been acquired by Oracle.

When you download Java, you get the JRE (Java Runtime Environment), which consists of the JVM (Java Virtual Machine), Java classes, and Java libraries. Any computer with the JRE can run Java applications, so Java is a "Write Once, Run Anywhere" language.

Java, C++, and C# all have similar syntax, so if you know one, it is easy to learn the others.
