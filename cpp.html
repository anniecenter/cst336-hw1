C++

In 1998, Danish software developer, Bjarne Stroustrup, created an extension to the C language and called it C++. C++ is a powerful, object-oriented programming language that allows programmers to work very closely with system resources and memory.

C++ is also a cross-platform language and is often used in operating systems, GUIs (Graphical User Interfaces), and embedded systems.

C++, Java, and C# all have similar syntax, so if you know one, it is easy to learn the others.
